ZORA — media assets
====================

Drop your real photos/video here and the site swaps them in automatically.
Until a file exists, the page shows a premium animated placeholder with a label,
so nothing ever looks broken.

Expected files (16:9 or larger, JPG/MP4):

HERO (homepage + about)
  hero-crowd.mp4     full-width background video loop (muted, ~10-20s, <8MB ideal)
  hero-crowd.jpg     poster/fallback still (shown before/without video)

FESTIVAL / EVENT IMAGERY (carousels + grids)
  event-01.jpg  event-02.jpg  event-03.jpg  event-04.jpg
  event-05.jpg  event-06.jpg  event-07.jpg  event-08.jpg
  artist-01.jpg artist-02.jpg artist-03.jpg
  crowd-01.jpg  crowd-02.jpg  crowd-03.jpg

Naming must match exactly. Recommended: 1600x900, optimized under ~300KB each.
